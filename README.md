# GROUPOMANIA FRONTEND
# WITH VUE/VUEX/ROUTER/VUETIFY/AXIOS

# SETUP
```
npm install -g @vue/cli

vue create Groupomania_Frontend

vue add router
vue add vuex
vue add vuetify
npm install axios
```
